# 💎 Luxe React Project

Luxe React Project is a modern beauty e-commerce web application built using React.js. It offers a smooth and elegant user experience where users can explore beauty products, watch tutorials, and manage their shopping activities with ease.

---

## 🚀 Features


- 🛍️ Browse and view product details
- ❤️ Add/remove items from Wishlist
- 🛒 Cart management with clean UI
- 🎥 Integrated beauty tutorial videos
- 🔄 Global state management using Context API
- 🌐 Navigation using React Router
- ⏳ Loading indicators for better UX
- 📦 Local JSON used as mock backend

---

## 🛠️ Tech Stack

- React.js  
- Context API  
- React Router  
- CSS  
- JSON (backend Data)
